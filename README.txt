# 취업 성공 가이드 (GitHub Pages)

## 업로드 방법
1) 이 폴더 안의 파일/폴더를 그대로 GitHub 리포지토리에 업로드
- index.html
- jobs.html
- companies.html
- detail.html
- data.json
- assets/ (styles.css, app.js)

2) GitHub Pages 설정
- Settings → Pages → Branch: main / (root) 저장

3) 접속
- https://<username>.github.io/<repo>/

## 데이터 수정
- 자료.xlsx를 수정한 뒤, data.json을 다시 생성해야 함.
